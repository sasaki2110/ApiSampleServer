## 2. 復習で確認した重要ポイント

- `ApiSampleServerApplication` の `main` は API 専用でも必要（Spring Boot 起動エントリ）
- `Map` 返却は疎通 API では実用的。業務 API は DTO（record など）が望ましい
- 「API 契約」は英語で `API contract`。OpenAPI は契約の明文化手段

## 4. Docker + フロント接続の運用メモ

- コンテナ内 `curl` では `host.docker.internal` が使える
- ただしブラウザ実行の `fetch` はホスト視点になるため、通常は `http://localhost:8080` を優先
- 詳細は `docs/08_フロント開発向け_最小接続実装.md` を参照
